Sobre Este Proyecto

Este proyecto fue desarrollado como parte del curso de Python y Django, con el objetivo de crear una plataforma de e-commerce completa para la venta de joyería en mayoreo.

Acerca de mi, comence este curso con la finalidad de aprender a programar en Python, ya que, era una habilidad necesaria para seguir creciendo en mi trabajo como ingeniero mecanico.

Tecnologías Utilizadas

Django 4.2+ • Python 3.13 • SQLite • HTML5 • CSS3 • Pillow • Class-Based Views • Function-Based Views • Authentication System

Características del Proyecto

• Sistema completo de gestión de productos con CRUD
• Autenticación y perfiles de usuario personalizados
• Manejo de imágenes con Pillow
• Búsqueda avanzada de productos
• Diseño responsive inspirado en Massimo Dutti
• Protección de vistas con decoradores y mixins
